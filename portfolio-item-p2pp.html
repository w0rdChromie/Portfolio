Yale play2Prevent Project (2022)

An interactive prevention plan.

This project was a collaboration between Yale's Play2Prevent lab and myself, with the purpose of using gameplay to create suicide prevention plans for its players. Similar to Figure It Out, this project was prototyped in the Twine Engine. Due to the nature of this project, and the type of audience testing it would require, I had to obtain biomedical research certifications from the U.S. Department of Health and Human Services to inform its design and development process so that players' safety would be a priority during their experience. The game uses a branch-like decision system as checkpoints that will help shape the prevention plan that would eventually be presented to the player. While this project was never completed in full, a general game system prototype can be played here.

    My contributions to this project were:
    • Prototyping builds in the Twine engine
    • Designing game navigation
    • Creating art assets in Photoshop and similar applications
    • Playtester
    • Facilitating playtesting sessions
    • Initial Research and Planning
