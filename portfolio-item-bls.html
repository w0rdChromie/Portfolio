Big Large Succ (2019)

A paint jam project.

This is a game our team made for the 2019 Paint Jam, a game jam where every asset had to be original and had to be made with minimal software use. We ended up using Microsoft Paint to make the visual assets and I ended up recording my voice for the sound effects. The game was made in Unity. The game can be only be played with the keyboard. You can download and play the game here (this will bring you to a new tab).

For this project I:

  • Provided some primitive Autodesk models for use in the game.
  • Provided audio and sound effects for the game.
  • Used my skills in audio design to modify my voice for sound effects.
