Figure It Out! (2020)

Viral outbreak prevention.

As an exercise in Mobile game-based learning, Figure It Out is meant to use the results of quantitative research to construct a thourough training experience for public health workers in sub-Saharan African countries. The game features a sole scenario that positions the user in the perspective of a key public health official and has them make key decisions that will influence their result which will be displayed at the end of the experience. A prototype was produced by multiple industry professionals alongside students like myself at the time, which can be played here.

    My contributions to this project were:
    • Prototyping builds in the Twine engine
    • Designing the keyword text indication system
    • Creating the core mechanic for the minigame featured in the playable build
    • Playtester
