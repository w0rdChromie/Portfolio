Symptom Selection (2020 - 2021)

A collaborative project.

This is game started out as a collaboration between a student from Quinnipiac's School of Medicine, two other game design students, and myself. The purpose of this game was to be used as a tool to help dermatology students learn their material in a more efficient manner. The game was made in Unity. You can play the game here. You can access the resulting poster for this project here. You can access the project abstract here (the links will bring you to a new tab).

For this project I:

  • Used C# to program the entire game.
  • Created auditory and visual assets to be used in the game.
  • Used my project management and teamwork skills to help finish this game.
