Symptom Selection (2020)

Enhancing the learning process.

This is game started out as a collaboration between a student from Quinnipiac's School of Medicine, two other game design students, and myself. The purpose of this game was to be used as a tool to help dermatology students learn their material in a more fun and efficient manner. The game was made in Unity. You can play the game here. You can access the resulting poster for this project here. You can access the project abstract linked here (the links will bring you to a new tab).

    My contributions to this project were:
    • Prototyping builds in the Unity engine
    • Designing menu navigation and minigames
    • Repository Management
    • Playtester
    • Soundtrack Design
