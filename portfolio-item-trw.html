The Right Way (2021)

Using games as commentary.

A friend and I worked on this game for our submission in Ludum Dare 48. The theme for Ludum Dare 48 was "Deeper and Deeper" so we decided that the game would involve the player going deeper into a basement with a bit of political commentary mixed in. The game was made in Unity. You can download and play the game here (the link will bring you to a new tab).

For this project I:

  • Performed QA and testing to ensure that the game wasn't broken.
  • Created auditory and visual assets to be used in the game.
  • Used my skills in teamwork to help finish this game in 48 hours.
