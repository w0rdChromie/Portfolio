Urban City Champions (2021)

Experimenting with 2D Fighters.

This is game was a collaborative effort between four Quinninipiac students, including myself. We wanted to try and make a 2D fighting game within the span of a semester in the hopes that development would continue afterwards. The game was made in Unity. You can download and play the game here (the link will bring you to a new tab).

For this project I:

  • Used C# to program a prototype build for testing purposes.
  • Performed QA and testing to ensure that the game felt mechanically sound.
  • Created visual assets to be used in the game.
  • Used my skills in teamwork to help finish this game.
